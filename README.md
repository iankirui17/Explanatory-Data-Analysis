# README: Mobile Money Fraud Detection Analysis

## Overview
This project analyzes trends in mobile money usage, fraud cases, and payment adoption patterns. The goal is to understand how digital financial activity changes over time and whether fraud levels rise alongside transaction growth.

## Requirements
Before running the analysis, ensure the following R packages are installed and loaded:

- dplyr
- tidyr

---

## Data Included
The analysis uses four datasets:

- Table2: Mobile users, number of transactions, and transaction value
- Table4: Reported fraud cases and estimated financial losses
- Table5: Adoption trends of various payment methods
- Table6: Mobile account usage trends among adults

---

## What the Analysis Does

- Calculates year-over-year growth for transactions, values, fraud cases, and estimated losses
- Displays the processed data in readable tables
- Creates visualizations showing:
    - Growth in mobile money transaction value
    - Comparison of digital payment method usage over time

---

## Output

Running the script will produce:

- Printed analysis tables
- Line charts showing key financial and fraud trends

---

## Purpose

This analysis supports fraud research by:

- Showing how mobile transactions have evolved
- Highlighting proportional increases in fraud reports and losses
- Helping identify patterns relevant to fraud detection and prevention
